## Restaurant page

Tenth project for TOP: Restaurant Page.   
Source code in `./src`, builds to `./dist`
.
### Commands

Run for development, open local dev server and watch for changes:

`
npm run dev
`

Build all resources from `./src` to `./dist`:

`
npm run build
`
